## Pygame 简介

**Pygame** 是一组用于开发**游戏软件**的 **Python 模块**，它建立在 **SDL (Simple DirectMedia Layer)** 库的基础上。

### 核心特点：

* **游戏开发框架：** 它允许您在 Python 程序中创建功能丰富的**游戏**和**多媒体程序**。
* **跨平台和高可移植性：** Pygame 是一个高可移植性的模块，可以支持包括 Windows、macOS 和 Linux 在内的**多个操作系统**。
* **包含图形和声音：** Pygame 封装了 SDL 库，提供了处理**图像、声音**、输入设备（如键盘、鼠标、游戏手柄）以及定时器等功能，非常适合 **2D 游戏**的开发。
* **简化游戏逻辑：** 它的设计理念是让开发者能够将精力集中在**游戏逻辑本身**，而无需被 C 语言或汇编语言等底层语言的细节所束缚。

简而言之，如果您想使用 **Python** 语言来轻松、快速地制作一个 **2D 游戏**，Pygame 是一个非常流行和优秀的选择。

## Pygame 的用途

Pygame 主要用于：

1.  **2D 游戏开发：** 制作休闲游戏、平台游戏、解谜游戏等。
2.  **多媒体应用：** 创建简单的多媒体程序或交互式应用。
3.  **教育：** 作为一个学习编程和游戏开发概念的工具。
