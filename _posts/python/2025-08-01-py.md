# 列表（List）与基本操作

```python
fruits = ["apple", "banana", "cherry"]
fruits.append("mango")
print(fruits[1])  # banana
```

删除元素有三个方法

## 使用 remove()：按值删除

```
fruits.remove("banana")
```

## 使用 pop()：按索引删除

```py
fruits.pop(2)
# 删除索引为2的元素（"cherry"）
```

## 使用 del：也可删除指定索引

```py
del fruits[0]
# 删除索引为0的元素（"apple"）
```

## 切片（Slicing）

```py
print(fruits[1:3])
# ['banana', 'cherry']
```

## 遍历列表：for 循环

```py
for fruit in fruits:
    print(fruit)
```
