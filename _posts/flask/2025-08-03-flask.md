# 什么是flask?

使用 Python 编写的轻量级 Web 框架，它的特点是：

微框架（Minimal）：只有最基本功能，扩展性强

易于上手，适合初学者

模块化强，适合构建小到中型网站或 API 服务

flask = python + HTTP 请求 + 模板渲染 + 数据库 + 扩展插件

# 学习这门课程你们可以学到什么？

学习 Flask 可以掌握 Web 后端开发核心技术，

包括路由设计、处理 HTTP 请求、模板渲染、数据库操作（CRUD）、用户认证、表单处理、文件上传、邮件发送、
项目结构化设计，以及将应用部署到服务器等技能，能够独立开发完整的网站与 API 服务
