大家好，我是python网页后端flask的讲师geo

- GET 请求：从服务器获取数据（通过 URL 查询参数传递）。
- POST 请求：将数据发送到服务器进行处理（如表单提交）。

```py
from flask import Flask, request, render_template

app = Flask(__name__)

@app.route("/", methods=["GET", "POST"])
def form_example():
    if request.method == "POST":
        name = request.form.get("name")  # 获取表单输入
        return f"Hello, {name}!"
    return render_template("form.html")  # 显示 HTML 表单

if __name__ == "__main__":
    app.run(debug=True)
```

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Form Example</title>
</head>
<body>
    <h1>Enter Your Name</h1>
    <form method="post">
        <input type="text" name="name" placeholder="Your Name">
        <button type="submit">Submit</button>
    </form>
</body>
</html>
```

### 实作

创建一个表单，输入一个数字 number，并在表单提交后返回该数字的立方值。

```py
from flask import Flask, request, render_template

app = Flask(__name__)

@app.route("/", methods=["GET", "POST"])
def form_example():
    if request.method == "POST":
        name = request.form.get("name")  # 获取表单输入
        number = request.form.get('number')
        return f"Hello, {name}! Your {number} is {int(number)**3}" # 可以撰写这个输出的html
    return render_template("form.html")  # 显示 HTML 表单

if __name__ == "__main__":
    app.run(debug=True)
```

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Form Example</title>
</head>
<body>
    <h1>Enter Your Name</h1>
    <form method="post">
        <input type="text" name="name" placeholder="Your Name">
        <input type="text" name="number" placeholder="Your Number">
        <button type="submit">Submit</button>
    </form>
</body>
</html>
```
