重定向是将用户从一个 URL 自动转到另一个 URL

Flask 使用 `redirect()` 函数来实现重定向。

```python
from flask import Flask , redirect , url_for

app = Flask(__name__)

@app.route('/')
def home():
    return "Hello, Flask!"

@app.route('/go')
def go():
    return redirect('/')

if __name__ == '__main__':
    
    app.run(debug=True)
```
