路由定义如何将 URL 映射到视图函数。

动态路由示例 :

```py
from flask import Flask

app = Flask(__name__)

@app.route("/")
def hello_world():
    return "I\'m geo"

@app.route("/user/<username>")
def show_user(username):
    return f"Hello, {username}!"

if __name__ == "__main__":
    
    app.run(debug=True)
```

任务 : 

创建一个动态路由 /square/<int:number>，返回数字的平方。

```py
from flask import Flask

app = Flask(__name__)

@app.route("/")
def hello_world():
    return "I\'m geo"

@app.route("/user/<username>")
def show_user(username):
    return f"Hello, {username}!"

@app.route("/square/<int:number>")
def square(number):
    return f'{number**2}'

if __name__ == "__main__":
    
    app.run(debug=True)
```
